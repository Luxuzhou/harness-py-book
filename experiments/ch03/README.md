## 两个实验的分工

- **exp1（演示性）**：来源于公众号文章《200 行 Python，三轮博弈》。
  自带一套独立的 mini-Harness（不依赖 harness_py 主框架），结构清晰、易读、
  对应 3.3 节叙事。读者跑完三种 `run_*.py` 能直观看到"防御层逐步生效"的过程。
  原位置 `examples/ch03_harness_demo/`，2026-04 迁入本目录。

- **exp2（统计性）**：把 exp1 的"演示性结论"升级为可统计的数据。固定 10 条攻击集
  分类覆盖（路径遍历 / 危险命令 / 内容泄漏），4 档防御配置（L0/L1/L2/L3）×
  3 seeds，输出每档拦截率均值与置信区间。

> 二者关系类似 Ch4 工具描述 eval：先有"看一眼就懂"的小 demo，再有"严格统计"
> 的回归测试。书稿正文同时引用，前者用作概念展示，后者用作数据支撑。
