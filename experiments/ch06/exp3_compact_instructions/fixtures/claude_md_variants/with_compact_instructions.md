# 项目约定

本项目是一个轻量 Agent 运行时的 Python 实现。以下约定适用：

- 所有源码使用 Python 3.10+
- 测试使用 pytest，文件命名 `test_*.py`
- 日志使用 `logging` 模块，不要用 `print`（输出结果除外）

## Compact Instructions

压缩对话历史时必须保留以下信息：

- **当前任务的完整目标和所有步骤**（特别是未完成的步骤清单）
- 正在修改的文件列表及修改原因
- 最近一次测试的结果
- 下一步计划

压缩摘要必须包含一个 `[Remaining Steps]` 段落，明确列出还未完成的步骤。
即使某些步骤已经被讨论过，只要 pytest 未全部通过，就应该视为未完成。
