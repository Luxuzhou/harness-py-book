# 项目约定

本项目是一个轻量 Agent 运行时的 Python 实现。以下约定适用：

- 所有源码使用 Python 3.10+
- 测试使用 pytest，文件命名 `test_*.py`
- 日志使用 `logging` 模块，不要用 `print`（输出结果除外）

## Compact Instructions

压缩对话历史时必须保留以下信息：

- 当前任务的完整目标（一句话描述即可）
- 用 `[x]` / `[ ]` 标记每个步骤的完成状态：
  - `[x] 步骤 N`：已完成，只保留标记，**不要重述步骤内容**
  - `[ ] 步骤 N`：未完成，保留步骤内容
- 正在修改的文件列表及修改原因
- 最近一次测试的结果（pass/total 格式）
- 进度游标：当前卡在哪一步的哪个子任务

压缩摘要的剩余步骤部分**只列未完成项**，不要把已完成的步骤再完整誊写一遍。
